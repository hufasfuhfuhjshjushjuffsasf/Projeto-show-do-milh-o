1-Pontuação:
-Exemplo: pergunta vale 64 mil, se parar você fica com a pontuação atual, se acertar ganha 64 mil e vai pra proxima pergunta, se errar ganha metade do valor anterior.

2-Dificuldade:
-As perguntas vão aumentando a dificuldade equivalente ao valor.

3-Ajuda:
-Você pode solicitar os convidados para pedir ajuda, mas cuidado só pode usar uma vez.
